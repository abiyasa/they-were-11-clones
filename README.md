# They were 11 Clones

## What's this?
* Originally made for [GitHub Game Off 2012 Challenge](https://github.com/github/game-off-2012)
* A 3D twin-stick shooter game on your browser using Adobe Flash!
* Clone yourself up to 11 clones!
